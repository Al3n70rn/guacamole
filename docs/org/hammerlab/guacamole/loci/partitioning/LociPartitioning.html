LociPartitioning

case class LociPartitioning(map: LociMap[PartitionIndex]) extends Saveable with TruncatedToString with Product with Serializable

A wrapper around a mapping from genomic-loci ranges to Spark partition numbers.

Includes some functionality for saving to / loading from disk.
