    Flatmap across loci, where at each locus the provided function is passed a Pileup instance.

    Flatmap across loci, where at each locus the provided function is passed a Pileup instance.

    skipEmpty

    If true, the function will only be called at loci that have nonempty pileups, i.e. those where at least one read overlaps. If false, then the function will be called at all the specified loci. In cases where whole contigs may have no reads mapped (e.g. if running on only a single chromosome, but loading loci from a sequence dictionary that includes the entire genome), this is an important optimization.

    the windowTaskFlatMapMultipleRDDs function for other argument descriptions
