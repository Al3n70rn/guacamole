InputFilters

case class InputFilters(mapped: Boolean = false, nonDuplicate: Boolean = false, passedVendorQualityChecks: Boolean = false, isPaired: Boolean = false) extends Product with Serializable

Filtering reads while they are loaded can be an important optimization.

These fields are commonly used filters. Setting a field to True will result in filtering on that field. If multiple fields are set, the result is the intersection of the filters (i.e. reads must satisfy ALL filters).

mapped

include only mapped reads

nonDuplicate

include only reads that do not have the duplicate bit set

passedVendorQualityChecks

include only reads that do not have the failedVendorQualityChecks bit set

isPaired

include only reads are paired-end reads
