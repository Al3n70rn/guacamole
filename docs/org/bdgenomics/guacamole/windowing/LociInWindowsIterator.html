LociInWindowsIterator

case class LociInWindowsIterator[Region <: HasReferenceRegion](ranges: Iterator[SimpleRange], skipEmpty: Boolean, windows: PerSample[SlidingWindow[Region]]) extends Iterator[PerSample[SlidingWindow[Region]]] with Product with Serializable

This iterator advances each per-sample SlidingWindow to the next locus.

Each call to next advances each per sample SlidingWindow one locus (unless skipEmpty is set, where it advances to the next locus where at least one window contains an element)

ranges

Collection of ranges the iterator should cover

skipEmpty

Skip loci if the windows contain no overlapping regions

windows

Sliding Window of regions

  59. def next(): PerSample[SlidingWindow[Region]]

    Next returns the sequences of windows we are iterating over, which have been each advanced to the next locus.

    Next returns the sequences of windows we are iterating over, which have been each advanced to the next locus. The window cover a region of (2 * halfWindowSize + 1) around the currentLocus and contain all of the elements that overlap that region

    Each call to next() returns windows that been advanced to the next locus (and corresponding overlapping reads are adjusted)

    returns

    The sliding windows advanced to the next locus (next nonEmpty locus if skipEmpty is true)
